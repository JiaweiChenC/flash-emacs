#+title: Flash Emacs

Flash Emacs provides flash.nvim-like jump navigation for Emacs.
It dynamically update the labels.

* Install

Add =flash-emacs.el= to your =load-path= and require it:

#+begin_src emacs-lisp
(require 'flash-emacs)
#+end_src

* Usage

#+begin_src emacs-lisp
M-x flash-emacs-jump
#+end_src

* Options

- =flash-emacs-labels=: characters used for labels.
- =flash-emacs-multi-window=: search across visible windows.
- =flash-emacs-case-sensitive=: nil, t, or smart.
- =flash-emacs-min-pattern-length=: minimum input length.
- =flash-emacs-auto-jump-single=: auto-jump on a single match.
- =flash-emacs-exclude-modes=: major modes excluded from search.
- =flash-emacs-max-buffer-size=: maximum buffer size for searching.
